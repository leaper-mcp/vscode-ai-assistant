# 侧边栏使用指南

## 📍 新增侧边栏功能

现在AI对话助手拥有了独立的侧边栏视图，提供更便捷的使用体验！

### 🔍 如何找到侧边栏

1. **活动栏图标**：在VSCode左侧活动栏中找到 💬 图标
2. **点击打开**：点击图标展开AI对话面板
3. **开始对话**：在底部输入框输入消息开始对话

### 🎨 界面布局

```
┌─────────────────────────┐
│      AI对话助手         │ ← 标题栏
├─────────────────────────┤
│ [配置] [清空]           │ ← 工具栏
├─────────────────────────┤
│                         │
│   📝 对话历史区域        │
│   • 用户消息            │
│   • AI回复              │
│                         │
│                         │
├─────────────────────────┤
│ [输入框____] [发送]     │ ← 输入区域
└─────────────────────────┘
```

### ✨ 主要特性

- **🎯 独立空间**：专用的对话面板，不干扰编辑器
- **💾 自动保存**：对话历史自动保存，重启不丢失
- **🎨 主题适配**：完美适配VSCode主题
- **⚡ 响应式设计**：侧边栏宽度自适应
- **🔄 实时更新**：消息实时显示，流畅体验

### 🛠 快捷操作

- `Ctrl+Alt+A`：快速打开AI对话侧边栏
- `Enter`：发送消息
- `Shift+Enter`：输入换行
- 点击"配置"按钮：快速访问设置
- 点击"清空"按钮：清除对话历史

### 📱 侧边栏优势

1. **更好的工作流**：边编码边对话，提高效率
2. **持续可见**：对话内容始终可见，便于参考
3. **空间优化**：不占用主编辑区域
4. **快速访问**：一键打开，即时使用

### 🎯 使用场景

- 💻 **编程辅助**：边写代码边询问技术问题
- 📚 **学习交流**：与AI讨论学习内容
- 📝 **文档编写**：实时获取写作建议
- 🔍 **问题排查**：遇到问题时快速询问
